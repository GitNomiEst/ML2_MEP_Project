Explanation of the model

The Python script, model.py, is responsible for training a machine learning model to classify asteroids based on their features.
Let's break it down the main functions:

load_neo_data(): Connects to a MongoDB database to fetch Near-Earth Object (NEO) data.
preprocess_data(): Processes the NEO data into a format suitable for training a machine learning model.
train_model(): Trains a Random Forest Classifier model using the preprocessed data.
evaluate_model(): Evaluates the trained model's accuracy using testing data.
save_feature_importance_plot(): Generates and saves a plot showing the importance of each feature in predicting asteroid hazard.
predict_danger(): Predicts the hazard level of a new asteroid using the trained model.
